## ⚙️ Installation & Setup
Clone the repo and run locally:

```bash
# Clone the repo
git clone https://github.com/HasinRafid/randomBlogClone1.git

# Enter project folder
cd randomBlogClone1

# Install dependencies
npm install

# Run development server
npm run dev
